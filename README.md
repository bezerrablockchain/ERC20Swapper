## ERC20Swapper

### .env

Use .env.example in order to setup the environment variables for this project

### Build

```shell
$ forge build
```

### Deploy

```shell
$ forge script script/ERC20SwapperUp.s.sol --rpc-url <your_rpc_url> --verify --etherscan-api-key <your_api_key> --broadcast 
```
